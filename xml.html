xml

xml

xml：Extensible Markup Language 可拓展标记语言。

功能：

1、作为配置文件；

2、在网络中传输。

xml与html的区别：

1、xml的标签可以自定义；

2、xml语法严格；

3、xml用来存储数据。

特点：

xml是不作为的。即xml不会去做任何事情，只是存储数据使用。

介绍

  1. 和html类似，xml是一种树型结构。每个节点都有自己的属性和内容。同时可能还有父节点与子节点。

  2. xml所有的元素都必须具有关闭标签。且标签对大小写敏感。

  3. xml必须正确的嵌套。

  4. xml标签可以有属性值，但是属性值必须加引号。

  5. xml中一些特殊字符必须有其实体引用，直接用这些字符会发生错误。如： 

                     &lt;    <    小于  
                 &gt;    >    大于  
                 &amp;    &    和号  
                 &apos;    '    单引号  
                 &quot;    "    引号

  6. xml注释写法为

  7. html中会把超过一个空格的其他空格自动删除，xml不会。

xml元素

xml元素中可以包含属性、文本以及其他元素。

xml元素命名规范

  1. 名称可以包含字母、数字和其他字符。

  2. 名称必须以字母开头。

  3. 名称不能以xml(包括其他大小写模式)开头。

  4. 名称中不能包含空格。

xml属性

  1. 属性值一般是不属于数据组成部分的信息，一般是对处理软件来说比较重要的信息。

  2. 属性值必须被引号包围。既可以是双引号，也可以是单引号。

  3. 在xml中，应该尽量避免属性。原因：

    1. xml中，属性不能拥有多重的值。

    2. 属性不好描述xml的树形结构。

    3. 属性相较于元素，不易拓展。

xml规则

  1. 第一行是 XML 声明。它定义 XML 的版本 和所使用的编码。如： <?xml version="1.0" encoding="ISO-8859-1"?>

  2. xml文档必须包含根元素，他是所有其他元素的父元素。

约束

DTD

DTD是一种简单的约束。

DTD 的作用是定义 XML 文档的结构。它使用一系列合法的元素来定义文档结构

创建DTD约束文档的方式：

内部DOCTYPE声明

在xml文档开头，定义格式:<!DOCTYPE 根元素 [元素声明]>

外部文档声明

本地dtd文档：

在外部文件创建一个.dtd文件，文件中撰写规则和上面的中括号中的元素声明一致。文件名称为根元素名称。

在xml文档中声明：<!DOCTYPE 根标签名 SYSTEM "dtd文档位置">来表示

网络dtd文档： <!DOCTYPE 根标签名 PUBLIC "dtd文档名字" "URL">

元素声明代表着这个xml文件应该拥有什么元素以及构建出该xml文档的树状结构。

缺陷

无法约束元素内容或元素属性。

scheam

使用规范

在xml文档中插入代码片段：

<note xmlns="http://www.w3school.com.cn" xmlns:命名前缀="命名空间" xsi:schemaLocation="命名空间 .xsd文件路径">

在xml标签前面加命名前缀:可以表示这个标签遵循该前缀所表示的scheam规范文档。当只有一个scheam文档时，可以省略。

JSOUP解析器

使用方法

1、导入jar包；

2、获取document对象；

3、获取对应的element对象；

4、获取数据。

具体步骤

  1. 获取Document对象：

    1. 使用Jsoup类中的parse方法获取Document对象。Jsoup.parse(File file "编码方式");这种方式用来获取本地的xml文件。

    2. 使用Jsoup类中的parse方法的一种重载方式获取Document对象。即Jsoup.parse(URL url，Int timeoutMiles);这种方式通常用来获取网页的html文件。

    3. 几点注意事项

      1. 在使用第一种方法时，如果.xml文件和class文件在同一个文件夹下，可以通过以下方式获取路径。

        1. 利用反射机制，获取类加载器对象。类名.class.getClassLoader();
        2. 使用类加载器中的getResource(String path)方法获取xml文件的地址，返回值类型为URL。
        3. 将获取到的URL地址数据调用getpath()方法，获取其地址字符串。
        4. 创建File对象。

      2. 在使用第二种方法时，URL可以通过直接创建的方式获取。使用new URL(String path)这个构造方法。参数传递进去的即是网页的网址。timeoutMiles参数传递进的是等待时间。即当超过等待时间仍然没有获取返回的数据时，抛出java.net.ConnectException异常。单位是毫秒。

  2. 通过Document对象，获取属性集。这里有以下几种方法：

    1. getElementsByTag()通过标签名获取Elements对象集。

    2. getElementsById()通过id值获取Elements对象集。

    3. getElementsByAttributeValue()根据属性名以及该属性的值获取Elements对象集。

  3. 通过Elements对象获取单个Element对象。具体实现方法和Document对象的方法一致。

  4. 调用Element对象的方法，获取值。

    1. text()，返回这个标签下的文本值。如果该标签下仍然存在子标签，则返回所有子标签的文本，以空格分隔。

    2. html()，返回该标签的子树。

  5. selector选择器，通过这个选择器可以获取到指定位置的元素。使用方法：

    1. 调用document对象的方法select()获取元素。

    2. 参数格式：标签名[属性名 = 值]。后面可以通过特定格式拼接字符串。

    3. 关系选择器Combinators

      Pattern | Matches | Example --- | --- | ---| E F | 由E元素衍生而来的F元素 | div a, .logo h1 E > F | F是E的直接子结点 | ol > li E + F | 一个F元素，紧接在E的前面 | li + li, div.head + div E ~ F | 在F元素前面加上E | h1 ~ p E, F, G | 所有匹配元素E F G a | [href], div, h3

代码实例
import org.jsoup.Jsoup;
import org.jsoup.nodes.Document;
import org.jsoup.nodes.Element;
import org.jsoup.select.Elements;

import java.io.File;
import java.io.IOException;
import java.net.URL;

public class jsoup {

    public static void main(String[] args) throws IOException {
        String path = jsoup.class.getClassLoader().getResource("test.xml").getPath();
        Document document = Jsoup.parse(new File(path), "utf-8");
        Elements name = document.getElementsByTag("person");
        System.out.println(name.size());
        for(Element s : name){
            System.out.println(s.text());
        }
        for(Element s : name){
            System.out.println(s.html());
        }
        Element e1 = document.getElementById("666");
        System.out.println(e1.text());
        Elements e2 = document.getElementsByAttributeValue("temp", "temp");
        System.out.println(e2.text());
        System.out.println("-------------");
        Elements e3 = document.select("person1[id=555] > name > xing + ming");
        System.out.println(e3.text());

    }
}
