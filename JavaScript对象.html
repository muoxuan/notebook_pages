JavaScript对象

JavaScript对象

function对象

定义方式

function 方法名 (参数列表){
    方法体
}

function对象的属性

length属性：参数的个数。

注意事项

1、参数列表中的参数不同写数据类型；

2、当有重名方法时，下面的方法会覆盖掉上面的方法(只要方法名相同就会覆盖掉)；

3、调用方法时，参数数量可以不等于参数列表中的参数数量。当调用时的参数数量少时，没有被赋值的参数为undefined。当调用时的参数数量多时，这些参数会被存进arguments数组中。(其他情况下这些参数也被存进该数组中)。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>
    <script>
    function f1(a,b) {
        document.write(a + "  " + b + "<br>");
    }
    f1(1,2);
    f1(3);
    function f2() {
        for(i = 0; i < arguments.length; i++){
            document.write(arguments[i] + " ");
        }
    }
    f2(1,2,3,4,5);


    </script>
</body>
</html>

网页输出： 

1 2
3 undefined
1 2 3 4 5

array对象

定义方式

1、var arr = new Array(数据列表);

2、var arr = [ 数据列表 ];

3、var arr = new Array(数组长度);

4、var arr = new Array();

属性

length: 数组的长度。

常用方法

join(""):返回一个字符串，包含这个数组中所有的元素，元素之间以参数的字符串分隔。

push:在字符串末尾添加一个元素，长度+1。

特点

1、可以访问任意的索引，如果超出范围，获取的结果是undefined。

2、数组长度可变，数据类型可变。当向超出范围的一个索引赋值时，数组长度变为该索引+1。

3、未被赋值索引，啥都不存储。获取这个值的时候也是啥都获取不到。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>

    <script>
    var a1 = new Array(1,2,3);
    var a2 = new Array(5);
    var a3 = [1,2];
    var a4 = new Array();
    document.write(a1 + "<br>");
    document.write(a2 + "<br>");
    document.write(a3 + "<br>");
    document.write(a4 + "<br>");
    document.write(a1.length + "<br>");
    document.write(a1[10] + "<br>");
    document.write(a1.length + "<br>");
    a1[10] = 6;
    document.write(a1 + "<br>");
    document.write(a1.length + "<br>");
    document.write(a3.join("*") + "<br>");
    a3.push(3);
    document.write(a3 + " " + a3.length);

    </script>
</body>
</html>

网页输出： 

1,2,3
,,,,
1,2

3
undefined
3
1,2,3,,,,,,,,6
11
1*2
1,2,3 3

Data对象

定义方式

var date = new Date();

常用方法

1、toLocaleString(),根据本地时间格式返回当前时间的字符串。

2、getTime(),返回当前时间距1970.1.1 00 : 00 的毫秒数。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>
    <script>

        var d = new Date();
        document.write(d + "<br>");
        document.write(d.toLocaleString() + "<br>");
        document.write(d.getTime());

    </script>
</body>
</html>

网页输出

Sat Nov 23 2019 19:24:40 GMT+0800 (中国标准时间)
2019/11/23 下午7:24:40
1574508280895

Math对象

无需定义Math对象，直接可以调用其属性值和方法。

属性

包含有数学中常用常数，如圆周率π：Math.PI，自然对数e：Math.E等等。

常用方法

Math.ceil(x),对x向上取整。

Math.floor(x),对x向下取整。

Math.round(x),对x进行四舍五入。

Math.random(),返回一个伪随机数，范围是[0,1)。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>
    <script>

        document.write(Math.PI + "<br>");
        document.write(Math.E + "<br>");
        document.write(Math.random() + "<br>");
        document.write(Math.ceil(5.5) + "<br>");
        document.write(Math.floor(5.5) + "<br>");
        document.write(Math.round(5.5) + "<br>");

    </script>
</body>
</html>

网页输出

3.141592653589793
2.718281828459045
0.3115209012221807
6
5

RegExp对象

正则表达式对象。

正则表达式的概念及语法：正则表达式

正则表达式对象的创建

2种创建方法：

1、var reg = new RegExp("正则表达式")；

2、var reg = /正则表达式/

常用方法

test()方法，用来判断参数是否匹配。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>
    <script>

        /*
        规定，密码必须是大于6位小于12位，且以字母开头，密码中必须包含大写字母、小写字母与数字。
         */
        var reg = new RegExp("^(?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])([a-zA-Z][a-zA-Z0-9]{7,11})$");
        var reg2 = /^(?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])([a-zA-Z][a-zA-Z0-9]{7,11})$/;
        document.write(reg.test("123455") + "<br>");
        document.write(reg.test("a1234567") + "<br>");
        document.write(reg.test("a1234567A") + "<br>");
        document.write(reg.test("A1234567aa") + "<br>");
        document.write(reg.test("a123456789123A") + "<br>");
        document.write(reg.test("a/123456A") + "<br>");
        document.write(reg2.test("123455") + "<br>");
        document.write(reg2.test("a1234567") + "<br>");
        document.write(reg2.test("a1234567A") + "<br>");
        document.write(reg2.test("A1234567aa") + "<br>");
        document.write(reg2.test("a123456789123A") + "<br>");
        document.write(reg2.test("a/123456A") + "<br>");

    </script>
</body>
</html>

Global对象

定义方式

全局对象，可以直接方法名()调用

常用方法

encodeURI(str)：URI编码。

decodeURI(uri)：URI解码。

encodeURIComponent(str)：URI编码，编码字符更多。

decodeURIComponent(str)：URI解码。

parseInt(str)：将字符串转换为数字类型。规则，从第一个字符开始，是数字类型的便转换，一直到碰到非数字类型。如果第一个不是数字，那么返回NaN。

isNaN()：判断是否为NaN，NaN有一个特点：不等于任何数据，包括NaN。

代码演示

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

</head>
<body>
    <script>

        var str = "abc123测试";
        var uri = encodeURI(str);
        document.write(uri + "<br>");
        document.write(decodeURI(uri) + "<br>");
        uri = encodeURIComponent(str);
        document.write(uri + "<br>");
        document.write(decodeURIComponent(uri) + "<br>");

        document.write(parseInt(str) + "<br>");
        str = "123aaa";
        document.write(parseInt(str) + "<br>");
        uri = NaN;
        document.write((uri ==NaN)  + "<br>");
        document.write((uri === NaN) + "<br>");
        document.write((isNaN(uri)) + "<br>");

    </script>
</body>
</html>

网页输出

abc123%E6%B5%8B%E8%AF%95
abc123测试
abc123%E6%B5%8B%E8%AF%95
abc123测试
NaN
123
