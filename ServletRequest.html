ServletRequest

ServletRequest

ServletRequest(接口) --> HttpServletRequest(接口) --> org.apache.catalina.connector.RequestFacade(实现类，由tomcat提供)。

首先当用户发送请求信息后，tomcat接收信息，并创建类对象。将发送信息封装为ServletRequset对象(多态),传递进service方法。即service方法中的servletRequest参数即为请求信息。

我们可以通过以下的几个方法来获取这个请求信息。

请求行数据

例如该请求行： GET /httptest_war_exploded/test?username=aaa&yes=提交 http/1.1

常用方法

1、获取请求方式：GET 

String    getMethod();

2、获取虚拟目录： /httptest_war_exploded

String    getContextPath();

3、获取Servlet地址：/th 

String    getServletPath();

4、获取get方式的请求参数 username=aaa&yes=提交 

String    getQueryString();

5、获取请求的URL：/httptest_war_exploded/test?username=aaa&yes=提交 

String    getRequestURI();  ///httptest_war_exploded/th

StringBuffer getRequestURL() //http://localhost/httptest_war_exploded/th

6、获取协议及版本：http/1.1 

String getProtocol();

7、获取客户机的ip地址 

String getRemoteAddr();

代码演示


import javax.servlet.annotation.ServletSecurity;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

@WebServlet("/th")
public class TestHttpSrevlet extends HttpServlet {
    @Override
    public void doPost(HttpServletRequest request, HttpServletResponse response){

    }
    @Override
    public void doGet(HttpServletRequest request, HttpServletResponse response){

        System.out.println("getMethod(): " + request.getMethod());
        System.out.println("getContextPath(): " + request.getContextPath());
        System.out.println("getServletPath(): " + request.getServletPath());
        System.out.println("getQueryString(): " + request.getQueryString());
        System.out.println("getRequestURI(): " + request.getRequestURI());
        System.out.println("getRequestURL(): " + request.getRequestURL());
        System.out.println("getProtocol(): " + request.getProtocol());
        System.out.println("getLocalAddr(): " + request.getLocalAddr());

    }
}

请求头数据

常用方法

获取所有的请求头名称： 

Enumeration\<String> getHeaderNames()；

根据请求头名称获取请求头数据： 

String getHeader(String name);

Enumeration操作方法和迭代器一致，如：hasMoreElements()方法，等同于hasNext()方法;nextElement()方法，等同于next()方法。

代码演示

import javax.servlet.annotation.ServletSecurity;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.util.Enumeration;

@WebServlet("/th")
public class TestHttpSrevlet extends HttpServlet {
    @Override
    public void doPost(HttpServletRequest request, HttpServletResponse response){

    }
    @Override
    public void doGet(HttpServletRequest request, HttpServletResponse response){
        Enumeration<String> headerNames = request.getHeaderNames();
        while(headerNames.hasMoreElements()){
            String temp = headerNames.nextElement();
            System.out.print(temp + "--");
            System.out.println(request.getHeader(temp));
        }

    }
}

import javax.servlet.annotation.ServletSecurity;

    }

其他功能

获取参数的通用方式

根据参数名称获取参数值： 

String getParameter(String name);

根据参数名称返回参数数组：一般为复选框 

String[] getParameterValues(String name);

返回所有的参数名称： 

Enumeration<String> getParameterNames();

获取所有参数名称和参数值的Map集合： 

Map<String String[]> getParameterMap();

代码演示

import javax.servlet.annotation.ServletSecurity;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.BufferedReader;
import java.io.IOException;
import java.util.Arrays;
import java.util.Enumeration;
import java.util.Map;

@WebServlet("/th")
public class TestHttpSrevlet extends HttpServlet {
    @Override
    public void doPost(HttpServletRequest request, HttpServletResponse response) throws IOException {
        String password = request.getParameter("password");
        System.out.println("getParamaster(): password = " + password);
        String [] passwords = request.getParameterValues("password");
        System.out.println("getParameterValues: passwords = " + Arrays.toString(passwords));
        System.out.println("getParameterNames() & getParameterMap()");
        Enumeration<String> e = request.getParameterNames();
        Map<String,String[]> m = request.getParameterMap();
        while(e.hasMoreElements()){
            String temp = e.nextElement();
            System.out.println(temp + " " + Arrays.toString(m.get(temp)));
        }


    }
    @Override
    public void doGet(HttpServletRequest request, HttpServletResponse response){

    }
}

请求转发

request.getRequestDispatcher(path)获取请求转发对象。

path为路径，即其他class配置的路径。

使用该对象的foward()方法便可以转发对象。参数为request,response。

请求转发的特点：

1、浏览器地址不变；

2、只能在服务器内部转发；

3、浏览器只发送一次请求。

共享数据

域对象：表示有一定作用范围的对象。可以再一定范围内共享。

request域，表示在一次请求的范围，一般用于请求转发的多个组员共享数据。

方法：setAttribute(String name, Object obj); 存储数据

getAttribute(String name); 获取数据

removeAttribute(String name); 通过键来移除键值对。
