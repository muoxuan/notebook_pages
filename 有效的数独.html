有效的数独

有效的数独

题目

判断一个 9x9 的数独是否有效。只需要根据以下规则，验证已经填入的数字是否有效即可。

数字 1-9 在每一行只能出现一次。

数字 1-9 在每一列只能出现一次。

数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。

数独部分空格内已填入了数字，空白格用 '.' 表示。

示例 1: 

输入:  
[
  ["5","3",".",".","7",".",".",".","."],
  ["6",".",".","1","9","5",".",".","."],
  [".","9","8",".",".",".",".","6","."],
  ["8",".",".",".","6",".",".",".","3"],
  ["4",".",".","8",".","3",".",".","1"],
  ["7",".",".",".","2",".",".",".","6"],
  [".","6",".",".",".",".","2","8","."],
  [".",".",".","4","1","9",".",".","5"],
  [".",".",".",".","8",".",".","7","9"]
]

输出: true

示例 2: 

输入:
[
  ["8","3",".",".","7",".",".",".","."],
  ["6",".",".","1","9","5",".",".","."],
  [".","9","8",".",".",".",".","6","."],
  ["8",".",".",".","6",".",".",".","3"],
  ["4",".",".","8",".","3",".",".","1"],
  ["7",".",".",".","2",".",".",".","6"],
  [".","6",".",".",".",".","2","8","."],
  [".",".",".","4","1","9",".",".","5"],
  [".",".",".",".","8",".",".","7","9"]
]
输出: false

解释: 除了第一行的第一个数字从 5 改为 8 以外，空格内其他数字均与 示例1 相同。 

 但由于位于左上角的 3x3 宫内有两个 8 存在, 因此这个数独是无效的。

说明:

  • 一个有效的数独（部分已被填充）不一定是可解的。
  • 只需要根据以上规则，验证已经填入的数字是否有效即可。
  • 给定数独序列只包含数字 1-9 和字符 '.' 。
  • 给定数独永远是 9x9 形式的。

通过代码

class Solution {
    public boolean isValidSudoku(char[][] board) {

        for(int i = 0; i < 9; i++){
            int [] kind1 = new int [10];
            int [] kind2 = new int [10];
            for(int j = 0; j < 9; j++){
                if(board[i][j] != '.'){
                    int n = ((int) board[i][j]) - 48;

                    if(kind1[n] == 0){
                        kind1[n]++;
                    }
                    else{
                        return false;
                    }
                }
                if(board[j][i] != '.'){
                    int n = ((int) board[j][i]) - 48;
                    if(kind2[n] == 0){
                        kind2[n]++;
                    }
                    else{
                        return false;
                    }
                }
            }
        }
            for(int i = 0; i < 9; i = i + 3){
                for(int j = 0; j < 9; j = j + 3){
                    int [] kind = new int[10];
                    for(int num1 = 0; num1 < 3; num1++){
                        for(int num2 = 0; num2 < 3; num2++){
                            if(board[i + num1][j + num2] != '.'){
                                int n = ((int) board[i + num1][j + num2]) - 48;
                                if(kind[n] == 0){
                                    kind[n]++;
                                }
                                else{
                                    return false;
                                }
                            }
                        }
                    }
                }
            }
            return true;


    }
}

思路

首先判断一排或一列中有无重复元素

创建两个长度为10的数组，分别命名为kind1、kind2. 其下标范围恰好为0 ~ 9。

然后2次循环来遍历数组，第一个for循环代表着每排或每列。第二个for循环遍历着这一排或一列的每个元素。如果该元素不是‘.’， 那么就是数字。将其转换为int型数据，即用（int）强制类型转换。转换后为该数字的ASCⅡ码。所以要将结果减去48来获得原本的数字。

然后令n = 该数字，判断kind1[ n ] 或 kind2 [ n ]是否为0，如果是则++，如果不是直接返回false。

在第二层循环外边初始化kind1和kind2。

然后判断每一个3x3的小矩阵中是否有重复元素。

同样是2层循环，分别表示为行与列，但是该循环每次+3，即将大矩阵分割为9块小矩阵，并依次遍历这9个小矩阵。

循环内层又有2层循环，用来遍历小矩阵中的行与列。

然后步骤同对整个矩阵的整行整列判断。但此时因为每个小矩阵为一个单位，所以应在内2层循环结束后初始化记录次数的矩阵。

注意事项：

1、 可以在一次循环内分别判断行与列，只需将2个循环的变量顺序颠倒即可，即board [ i ] [ j ] 、 board [ j ] [ i ]。

2、在最后不要忘记return true。
