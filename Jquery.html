Jquery

Jquery

Jquery是JavaScript的一个框架。

使用步骤

  1. 下载Jquery

    1. Jquery-xxx.js：开发版本，给程序员看的。有良好的缩进与注释。

    2. Jquery-xxx.min.js 生产版本，程序中使用。没有缩进与注释。体积较小，程序加载更快。

  2. 在项目中导入Jquery

  3. 使用

Jquery对象与JavaScript对象的转换

两者获取的对象都可以当做数组来使用。

但是Jquery获取的对象使用更加方便。

例如，想要将同一个标签的所有内容全部变更。JavaScript对象需要使用for循环依次遍历。而Jquery对象可以使用其html方法做到这一点。

Jquery对象与JavaScript对象方法不通用。

所以需要Jquery对象与JavaScript对象转换。

JavaScript --> Jquery: $(JavaScript对象)。

Jquery --> JavaScript：jq对象[索引]或者jq对象.get(索引)。

Jquery事件绑定与入口函数

事件绑定类似java的匿名内部类。

入口函数： 

$(function(){

});

当dom文档加载好之后执行该函数。

window.onload只能调用一次，Jquery的入口函数可以定义多次。

Jquery对象同样可以控制css样式。利用其内部的css方法来进行修改。使用键值对的方式进行修改。

Jquery选择器

  1. 基本选择器

    1. 标签选择器：$("html标签名")

    2. id选择器：$("#id的属性值")

    3. 类选择器：$(".class的属性值")

    4. 并集选择器：$(选择器1,选择器2....) 选取多个选择器选取元素的并集。

  2. 层级选择器

    1. 后代选择器：$("A B ")选择A元素内部的所有B元素。(A下所有的B)

    2. 子选择器：$("A > B")选择A元素内部的所有B子元素。(A下的子元素)

  3. 属性选择器

    1. 属性名称选择器：$("a[属性名]")包含指定属性的元素。

    2. 属性选择器：$("a[属性名 = '值']")包含指定属性等于指定值的元素。

    3. 复合属性选择器：$("a[属性名 = '值'][][]....")

  4. 过滤选择器

    1. 首元素选择器：first 获取选择的元素中的第一个元素。

    2. 尾元素选择器：last 获取选择的元素中最后一个元素。

    3. 非元素选择器：not(selecter) 不包含指定内容的元素。

    4. 偶数选择器：even 偶数，从0开始计数。

    5. 奇数选择器：odd 奇数，从0开始计数。

    6. 等于索引选择器：eq(index) 指定索引元素。类似的还有大于索引选择器gt和小于索引选择器lt。

    7. 标题选择器：header 获取标题元素。

  5. 表单过滤选择器

    1. 可用元素选择器：enabled 获取可用元素。

    2. 不可用元素选择器：disabled 获取不可用元素。

    3. 选中选择器：checked 获取单选(复选)框选中的元素。

    4. 选中选择器：selected 获取下拉框选中的元素。

Jquery的DOM操作

内容操作

  1. html()：获取或设置元素标签体内容。

  2. text()：获取或设置元素标签纯文本内容。

  3. val()：获取或设置元素的value值。

属性操作

  1. 通用属性操作

    1. attr()：获取或设置元素的属性：元素固有的属性

    2. removeattr()：删除属性

    3. prop()：获取或设置元素的属性：元素自定义属性

    4. removeprop()：删除属性

  2. 对class属性的操作

    1. addClass()：添加class属性值

    2. removeClass()：删除class属性值

    3. toggleClass()：切换class属性：如果元素对象上存在这个class，就将其删除，如果不存在，就将其添加。

  3. CRUD的操作

    1. append()：父元素将子元素追加到末尾。例如：A.append(B)将B追加到A的元素内部的结尾。

    2. prepend()：父元素将子元素追加到开头。

    3. appendTo()：append方法的方向相反。

    4. prependTo()：prepend方法的方向相反。

    5. after()：添加元素到对应的元素后边。 A.after(B)将B添加到A后边

    6. before()：添加元素到对应的元素前边。

    7. insertAfter()：after方法的方向相反。

    8. insertBefore()：before方法的方向相反。

    9. remove()：将对象删除。

    10. empty()：清空元素的所有子元素。

JQuery动画

  1. 默认显示和隐藏方式：

    1. show([speed,[easing],[fn]])

    2. hide([speed,[easing],[fn]])

    3. toggle([speed,[easing],[fn]])

  2. 滑动显示和隐藏方式：

    1. slideDown([speed,[easing],[fn]])

    2. slideUp([speed,[easing],[fn]])

    3. slideToggle([speed,[easing],[fn]])

  3. 淡入淡出显示和隐藏方式：

    1. fadeIn([speed,[easing],[fn]])

    2. fadeOut([speed,[easing],[fn]])

    3. fadeToggle([speed,[easing],[fn]])

  4. 参数：

    1. speed：速度，三个预定义的值(slow,normal,fast)，或设定动画时长的毫秒值。

    2. easing：指定切换效果，默认是swing(先慢再快再慢)，可用参数：linear：匀速。

    3. fn：动画执行完毕后调用的函数。

JQuery遍历

  1. JQ对象.each(callback)

  2. JQuery.each(objeck,[callback])

  3. for..of

callback:回调函数。获取遍历到的对象：

  1. this关键字。

  2. 在回调函数中定义参数index，element。index：对象的索引，element元素对象。

break：结束循环。

continue：结束本次循环，进入下次循环。

for ... each JQuery3.0之后提供的方法。

for(li of city){};

JQuery事件绑定

  1. JQu标准的绑定方式：JQ对象.事件方法(回调函数);

  2. on的方式绑定事件，off的方式解除绑定。JQ对象.on(“事件名称”，回调函数); JQ对象.off(“事件名称”);

  3. 事件切换：toggle：JQ对象(fn1,fn2...)每点一下切换一次事件。
