查询表中记录

查询表中记录

基础查询

查询多个字段

SELECT 查询的字段1，查询的字段2... FROM 表名;

如果想要查询所有字段，可以使用*来代替。

查询不重复字段

SELECT DISTINCT 查询的字段1，查询的字段2... FROM 表名;

只有被查询的字段完全相同才会被忽略。

查询中可插入四则运算

SELECT 字段1 运算符 字段2 别名 FROM 表名;

别名指的是在查询出结果之后，将列名暂时变为别名。

IFNULL语法：

判断当数据为NULL时，可以将这个NULL替换为别的数据。

IFNULL(可能出现NULL的列名, NULL替换为的值);

条件查询

语法：SELECT 字段 FROM 表名 WHERE 限制条件;

限制条件运算符：

< ， > , = , <= , >= , <> (不等于)

与或非：AND,OR,NOT

BETWEEN ... AND ... 在...范围之内。

判断是否是NULL不能使用= 和 <>, 需要使用IS 与IS NOT

例如： 

SELECT * FROM TEST WHERE AGE BETWEEN 20 AND 40 OR SCORE IS NOT NULL;

LIKE 模糊查询：

占位符：_任意单个字符；%任意多个字符。

例如： 

SELECT * FROM TEST WHERE NAME LIKE 'A%';

排序查询

语法：

SELECT 字段 FROM 表名 ORDER BY 排序字段1 排序方式，排序字段2 排序方式... 排序字段n 排序方式;

注意事项：排序字段越往前，优先级越高。

排序方式有两种：ASC 升序排序，DESC降序排序。如果没填，则默认升序。

例如

SELECT * FROM TEST ORDER BY ID DESC,SCORE;

聚合函数

将一列数据作为一个整体，对其进行计算。

格式：SELECT 函数(列名) FROM 表名;

常用函数：
COUNT() 计算个数
MIN() 计算最小值
MAX() 计算最大值
SUM() 计算和
AVG() 计算平均值

注意事项：不会计算NULL值，遇到NULL值会忽略。

解决方法：1、使用IFNULL()语句将NULL替换为0；

2、选择不包含NULL值的列(主键)。

例如


SELECT COUNT(SCORE) FROM TEST;
SELECT MAX(SCORE) FROM TEST;
SELECT MIN(SCORE) FROM TEST;
SELECT AVG(SCORE) FROM TEST;
SELECT SUM(SCORE) FROM TEST;

分组查询

语法：SCLECT 被查询记录 FROM 表名 GROUP BY 分组列名。

被查询记录通常使用聚合函数。

例如：查询不同年龄的数量以及分数和。 

SELECT AGE,COUNT(ID),SUM(SCORE) FROM TEST GROUP BY AGE;

分组查询可以使用限定语句进一步限定分组情况： 

SELECT AGE,COUNT(ID) FROM TEST WHERE ID > 1  GROUP BY AGE HAVING COUNT(ID) < 2;

where和having的区别：

1、where是在分组之前进行限定，如果不满足条件则不将次记录进行分组；having是在分组后进行限定，不满足条件的分组不会被查询出来。

2、where后不能接聚合函数，having则可以。

分页查询

语法：SCLECT 被查询记录 FROM 表名 LIMIT 开始索引 显示记录条数;

例如： 

SELECT * FROM TEST LIMIT 0, 3;

SELECT * FROM TEST LIMIT 3, 3;
