JDBCtemplate

JDBCtemplate

Spring框架对JDBC进行了简单封装，其中提供了一个JDBCTemplate对象来简化操作。

使用步骤

  1. 导入JDBCTemplate的相关jar包。

  2. 创建数据源DateSource(可以通过使用数据库连接池来获取)

  3. 直接进行操作。

    1. 对表进行增删改DML操作：
      使用JDBCTemplate对选哪个中的Update方法进行操作。参数为：sql语句(String类型)、传递进的参数1、参数2...。返回值类型为受影响的行数。
    2. 对表中数据的查询。

使用queryForMap方法查询数据

queryForMap方法需求的参数和Update方法一致，直接传递进sql语句与参数即可。

返回值为一个Map集合。其中为查询的内容。

注意：该查询方式只能返回一行的数据，如果多于一行会抛出IncorrectResultSizeDataAccessException异常。

使用queryForList方法查询数据

queryForList方法需求参数与上面的方法一致。

返回值类型是一个List集合，泛型为为一个Map集合。即当查询结果为多条数据时，现将每一条数据封装进Map集合中，再讲该Map集合添加进List集合中。

使用query方法查询数据

query方法的参数列表与前两种方法不同，其需要sql语句及其参数，还需要一个RowMapper对象。

我们可以使用BeanPropertyRowMapper对象来替代RowMapper对象，前者为后者的实现类。

new一个BeanPropertyRowMapper对象，泛型为数据库的每行数据的一个自定义类。 参数为这个自定义类的.class对象。

返回值为这个自定义类对象的LIst集合。

这个自定义类中成员变量即为数据库的列名。查询到的词条会被传递进这个自定义类对象中。(注意：这个自定义类的成员变量必须是对象或者是基本数据类型的包装类。)

代码实例

自定义类

public class TEST1 {
    private Integer ID;
    private String NAME;
    private Integer CLASSID;

    public Integer getID() {
        return ID;
    }

    public String getNAME() {
        return NAME;
    }

    public Integer getCLASSID() {
        return CLASSID;
    }

    public void setID(Integer ID) {
        this.ID = ID;
    }

    public void setNAME(String NAME) {
        this.NAME = NAME;
    }

    public void setCLASSID(Integer CLASSID) {
        this.CLASSID = CLASSID;
    }

    @Override
    public String toString() {
        return "TEST1{" +
                "ID=" + ID +
                ", NAME='" + NAME + '\'' +
                ", CLASSID=" + CLASSID +
                '}';
    }
}

代码

import com.alibaba.druid.pool.DruidDataSourceFactory;
import org.springframework.jdbc.core.BeanPropertyRowMapper;
import org.springframework.jdbc.core.JdbcTemplate;

import javax.sql.DataSource;
import java.io.InputStream;
import java.util.List;
import java.util.Map;
import java.util.Properties;
public class DruidTest {
    public static void main(String[] args) throws Exception {
        InputStream is = DruidTest.class.getClassLoader().getResourceAsStream("druid.properties");
        Properties p = new Properties();
        p.load(is);
        DataSource ds = DruidDataSourceFactory.createDataSource(p);
        JdbcTemplate jt = new JdbcTemplate(ds);
        String sql1 = "UPDATE TEST1 SET NAME = ? WHERE ID = ?";
        jt.update(sql1,"777",1);
        String sql2 = "SELECT * FROM TEST1 WHERE ID = ?";
        Map<String, Object> m = jt.queryForMap(sql2, 1);
        System.out.println(m);
        String sql3 = "SELECT * FROM TEST1";
        List<Map<String, Object>> maps = jt.queryForList(sql3);
        for(Map<String,Object> ms : maps){
            System.out.println(ms);
        }
        System.out.println("--------------------");
        List<TEST1> l = jt.query(sql3, new BeanPropertyRowMapper<TEST1>(TEST1.class));
        for(TEST1 t : l){
            System.out.println(t);
        }
    }
}
